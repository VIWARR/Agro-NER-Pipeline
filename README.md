#AgroSifter — это высокопроизводительный pipeline для извлечения структурированных данных из неструктурированных сообщений агро-чатов Telegram. Проект решает проблему высокого уровня «шума» (спам, реклама, дубликаты) перед обработкой данных через LLM.

#Бизнес-задача
Оптимизация затрат на API DeepSeek за счет многослойной фильтрации входящего потока данных. Система превращает хаотичные сообщения в структурированную базу данных (товар, цена, сорт, регион).
